# rplayer
Terminal mp3 player written in Rust

Usage: ``` ./rplayer [path]```
## Shortcuts 

Shortcut                 | Action
-------------------------|--------
Ctrl + Left/Right arrow  |Move timestamp
 Left/Right arrow        |Set previous/next track
Alt + Left/Right arrow   |Skip 5 tracks and set track 
Space                    |Toggle pause
Up/Down arrow            |Control volume 
S                        |Shuffle
Esc                      |Close player
